## Secure, validate password

**Description**

- System shall only accept passwords with valid password structure

**Inputs**

- Password 

**Output**

- If input password is valid password or invalid password

**Action**

- System shall check if password have minimum 8 characters
- System shall check if password have any special characters (@#$!%^&*)
- System shall check if password have simple and capital letters
- System shall check if password have numbers

**Requirements**

- Password

**Side effects**

- None

**Pre-conditions and post-conditions**

- None
