## Define roles and rules that restrict and control access inside the dashboard.

**Description**

- System shall define roles to restrict access on parts of the dashboard.
- System shall define rules to restrict actions can be taken in dashboard with roles.
- System shall only give access to create, update, delete roles and rules only to user with super-user role.
