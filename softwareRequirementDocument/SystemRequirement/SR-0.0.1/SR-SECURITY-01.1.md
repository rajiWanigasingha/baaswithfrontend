## Secure, validate emails

**Description**

- System shall only accept emails with valid email structure

**Inputs**

- Email address

**Output**

- If input email is valid email or invalid email

**Action**

- Check if email have a structure similar to `name + @ + name + . + name`

**Requirements**

- Email

**Side effects**

- None

**Pre-conditions and post-conditions**

- None
