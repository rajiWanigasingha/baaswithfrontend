## Authorization and roles requirements

UR-AUTH-01 : Users shall be able to log in to the dashboard using an email and a password

UR-ROLE-02 : First user to log in to the dashboard shall be super-user of the dashboard

UR-ROLE-03 : Only super-user shall be able to create new users, delete users, modify user information for users in dashboard.

UR-ROLE-04 : Super-user shall be able to create rules other users to follow.

UR-ROLE-05 : Super-user should be able to create roles to restrict access, action that can be taken within dashboard.

UR-AUTH-06 : Users should be able to change there email and password.
